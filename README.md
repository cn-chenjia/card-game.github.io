### 第三步：战斗对决 ⚔️

#### 🔄 回合流程

```
┌─────────────────────────────────────────────┐
│           📍 每回合流程                       │
├─────────────────────────────────────────────┤
│  1️⃣ 抽卡阶段 → 双方各抽3张装备               │
│  2️⃣ 掷骰阶段 → 决定先手顺序                  │
│  3️⃣ 攻击阶段 → 攻方出牌攻击（最多3次）        │
│  4️⃣ 防御阶段 → 守方选择是否防御              │
│  5️⃣ 结算伤害 → 计算并扣除血量                │
│  6️⃣ 反击机会 → 守方可选择反击                 │
│  7️⃣ 回合结束 → 进入下一轮（共10轮）          │
└─────────────────────────────────────────────┘
```

#### 💥 伤害计算公式

```
最终伤害 = ⌈攻击值 × (1 + 攻击加成) - 防御值 × (1 + 防御加成)⌉

扣血量 = 最终伤害 ÷ 30

示例：使用倚天剑(28)攻击，对方用金丝宝甲(24)防御
→ 伤害 = 28 - 24 = 4点
→ 扣血 = 4 ÷ 30 = 0.13点HP
```

#### 💰 金币奖励

- ✅ **攻击成功造成伤害** → 获得 `伤害值 × 20` 金币
- 🛡️ **完美防御（伤害为0）** → 获得 `300` 金币
- 初始金币：**1000** 💰

## 🌟 特色系统

### 🎯 史诗技能系统

史诗级卡牌附带**强力被动技能**，可以在战斗中触发：

| 技能 | 图标 | 效果 | 触发时机 |
|------|------|------|----------|
| **偷取** | 🤚 | 偷取对方一张卡牌 | 攻击时 |
| **破防** | 💥 | 对方本次不能使用防御 | 攻击时 |
| **削弱** | ⬇️ | 对方防御值减少20% | 攻击时 |
| **降级** | 🔄 | 将对方一张史诗卡替换为普通卡 | 攻击时 |
| **封技** | 🚫 | 限制对方下次不能使用角色技能 | 攻击时 |

## 🎯 游戏技巧

1. **合理分配资源** 💰 — 不要把金币都花光，留一些应对紧急情况
2. **攻防平衡** ⚖️ — 只堆攻击容易被反杀，只堆防御打不动人
3. **善用史诗技能** 🌟 — 偷取和破防在关键时刻可以逆转局势
4. **针对克制** 🎯 — 根据对手的角色特点调整策略
5. **完美防御奖励** 🛡️ — 成功完全防御也能赚金币，不要忽视防具
6. **武学搭配** 📚 — 攻击型和防御型武学要配合角色的先天属性
